-Project Overview
This project is an interactive Excel Dashboard built using a Coffee Shop sales dataset. The goal of the project is to analyze sales performance, product trends, and revenue insights using advanced Excel functions and data visualization techniques.
The dashboard provides dynamic insights into sales, product performance, and business trends using formulas such as XLOOKUP and INDEX.

-Objectives
1.Analyze total sales and revenue trends
2.Identify best-selling products
3.Track category-wise performance
4.Monitor daily/monthly performance
5.Create an interactive dashboard for business decision-making

-Key Excel Functions Used
1.XLOOKUP
Used to dynamically fetch product details, prices, and sales values from structured tables.
Example:
=XLOOKUP(A2, Product_Table[Product_ID], Product_Table[Product_Name])

2.INDEX + MATCH
Used for flexible data retrieval and dynamic referencing.
Example:
=INDEX(Sales_Data[Revenue], MATCH(A2, Sales_Data[Product_ID], 0))
